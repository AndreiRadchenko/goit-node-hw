# Node.js CLI application

## Action exampes

- Recieve list of contacts and print it in table to console

```cli
node index --action list
```

![img](https://monosnap.com/file/mWhe8cp5Qt0AGaN38gwLSLsw0oJnBC)

<img src="https://monosnap.com/file/mWhe8cp5Qt0AGaN38gwLSLsw0oJnBC" width="100%"></img>

- Get contact by id

```cli
node index --action get --id 5
```

- Add contact

```cli
node index --action add --name Mango --email mango@gmail.com --phone 322-22-22
```

- Remove contact

```cli
node index --action remove --id 3
```
